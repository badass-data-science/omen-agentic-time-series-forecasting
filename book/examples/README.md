# Book Example Datasets and Plots

Every named dataset used in "Agentic Time Series Forecasting for Supervillains" — Secret
Lab™ Mojito Inventory, Death-Ray Revenue, Henchman Costume Dry-Cleaning
Bills, Secret Lab™ Power Consumption, and the rest — is generated by
`generate_book_datasets.py`, deterministically, from a fixed seed. Every
image embedded in the book — series plots, ACF/PACF, seasonal
decompositions, backtest/forecast/drift plots — is generated by
`generate_book_plots.py`, built on top of those same datasets and the
real `plot_*` tools this project ships. Every tool result and every
image in the book comes from running the real Omen tools against the
output of these two scripts, not from hand-typed, estimated, or
hand-drawn material.

## Regenerate everything

```bash
cd book/examples
python generate_book_datasets.py
python generate_book_plots.py
```

Writes every dataset as a CSV into `./data/`, and every plot as a PNG
into `./images/`. Requires Omen installed (`pip install -e ".[all]"`
from the project root — `omen.data_prep` itself only needs
`numpy`/`pandas`, but you'll want the rest to actually run the tools
each chapter calls; plotting needs `matplotlib`, a core dependency).

Regenerate just one dataset or plot:

```bash
python generate_book_datasets.py --only deathray_revenue
python generate_book_plots.py --only ch10_sarima_search
```

See the docstring on each function in `generate_book_datasets.py`/
`generate_book_plots.py` for which chapter(s) use that dataset or image
and why it's shaped the way it is.

## Why this file exists

Chapter drafts for this book are held to the same standard the rest of
the Omen project holds its own claims to: a number — or an image —
doesn't go in the book until it's been produced by a real run against
real data. That promise is only honest if a reader can rerun the same
generation and get the same series, and the same picture, back — which
is the entire job of these two scripts. (Confirmed directly: every
image `generate_book_plots.py` produces is currently byte-for-byte
identical to the one embedded in the book.)
